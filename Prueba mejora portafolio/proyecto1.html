Descripción del Proyecto

Este proyecto permite convertir monedas de diferentes países utilizando una API externa para obtener las tasas de cambio en tiempo real. Simple y fácil de usar.

Tecnologías usadas:

  • HTML
  • CSS
  • JavaScript
  • API de tasas de cambio
